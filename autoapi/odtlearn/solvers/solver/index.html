class odtlearn.solvers.solver.Solver(verbose: bool = False)[source]¶

Bases: abc.ABC

An abstract base class for interacting with the solver, including for setting up, optimizing, and getting optimal values of a model.

Parameters:
verbosebool, default=False

Whether to print verbose output from the solver.

Notes

This is an abstract base class and cannot be instantiated directly. Subclasses must implement the following abstract methods:

  • add_vars()

  • add_constr()

  • set_objective()

  • set_time_limit()

  • set_num_threads()

  • get_num_variables()

  • get_num_constraints()

  • optimize()

  • get_var_value()

  • get_gap()

  • get_objective_value()

  • get_objective_bound()

  • get_num_int_variables()

  • get_num_nonzeroes()

  • get_num_soluions()

  • get_search_progress_log()

  • lin_expr()

  • quicksum()

  • set_callback()

  • add_lazy_constraint()

  • get_callback_solution()

  • store_data()
