class trees.RobustTree.RobustTreeClassifier(depth=1, time_limit=1800, num_threads=None)[source]#

An optimal robust decision tree classifier, fitted on a given integer-valued data set and a given cost-and-budget uncertainty set to produce a tree robust against distribition shifts.
